# B207 - Cyber Security - Final Project - Secure Password Manager

A password manager built with Python that stores encrypted credentials in a MySQL database and
provides registration, login, password generation, and credential management
through a Flask web interface.

## Project Structure

```
.
├── app.py                    Flask web application: routes, sessions, forms
├── setup_environment.py      One-shot script that installs everything and sets up the database
├── requirements.txt          List of required Python packages
├── .env.example               Template of required settings, with no real secrets
├── .env (not committed)       Real database password and Flask secret key, generated by setup_environment.py
├── backend/
│   ├── auth.py                Registration and login
│   ├── crypto_utils.py        Hashing, key derivation, encryption/decryption
│   ├── db.py                  MySQL connection layer
│   ├── password_generator.py  Secure random password generation
│   └── vault.py               Vault entry CRUD
├── database/
│   └── schema.sql             Database schema
├── templates/                 Flask HTML templates
├── static/                    CSS and JavaScript
└── tests/
    └── manual_cli.py          Manual command-line test harness for the backend
```

## Uploading and Starting the Project

**Prerequisites:** Python 3.x and a running local MySQL server.

```bash
python setup_environment.py
```

This installs the Python dependencies, creates the database and tables,
creates a dedicated MySQL application user, and writes a `.env` file.

## Usage

```bash
python app.py
```

Open `http://127.0.0.1:5000` in a browser, register an account, log in, and
use the dashboard to add, generate, view, and delete vault entries.

## Manual Backend Testing

```bash
python -m tests.manual_cli
```
